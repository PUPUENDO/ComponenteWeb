Fundamentos del Desarrollo Web

HTML5 Semántico

El HTML semántico es la base de cualquier sitio web moderno. Utiliza etiquetas como <header>, <footer>, <article> y <section> para dar significado a la estructura del contenido.

<header>
    <h1>Título Principal</h1>
    <nav>
        <ul>
            <li><a href="#">Inicio</a></li>
            <li><a href="#">Acerca de</a></li>
        </ul>
    </nav>
</header>

CSS3 y Diseño Responsivo

CSS3 introduce características avanzadas como Flexbox, Grid, animaciones y media queries que permiten crear diseños adaptables a cualquier dispositivo.

Flexbox

Sistema de diseño unidimensional para distribuir espacio entre items

CSS Grid

Sistema de diseño bidimensional para layouts complejos

Media Queries

Adaptar estilos según características del dispositivo

JavaScript Moderno

JavaScript ha evolucionado enormemente con ES6+, introduciendo clases, módulos, arrow functions, async/await y más.

// Arrow function y template literals
const saludar = (nombre) => {
    return `Hola, ${nombre}!`;
};

// Async/await para manejo de promesas
async function obtenerDatos() {
    try {
        const respuesta = await fetch('api/datos');
        const datos = await respuesta.json();
        console.log(datos);
    } catch (error) {
        console.error('Error:', error);
    }
}
